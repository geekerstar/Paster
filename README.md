## 📋 功能特性

### 🗂️ 剪贴板历史记录
自动保存你复制的一切——文本、图片、文件、链接、代码，再也不怕覆盖丢失。

### 🔍 搜索过滤
实时搜索历史记录，快速定位所需内容，支持中文拼音模糊匹配。

### ❤️ 收藏置顶
重要内容一键收藏，始终置顶不被清除。

### ⌨️ 快速插入文本
按下 `Ctrl + 3` 呼出快速插入面板，从历史文本列表中一键选取并发送到当前输入框，无需粘贴覆盖。

### 🖱️ 拖拽贴图
剪贴板卡片支持直接拖拽到桌面上作为贴图，文字、图片、代码任意内容均可一键 Pin 到桌面，随时查看。

### 📷 截图工具
内置截图功能，支持：
- 全屏截图 / 矩形截图
- 椭圆标注 / 箭头标注 / 画笔涂鸦 / 文字标注
- 撤销 / 重做

### 📌 桌面贴图
截图可钉在桌面上，支持：
- 缩放 / 旋转 / 翻转
- 画笔标注 / 文字标注
- 随意拖动位置

### ⌨️ 全局快捷键
- `Ctrl + 1` — 呼出/隐藏主窗口（可自定义）
- `Ctrl + 2` — 启动截图（可自定义）
- `Ctrl + 3` — 快速插入文本（可自定义）

### 🌓 深色主题
明暗双主题一键切换，保护视力，适配不同使用环境。

### 🎨 卡片自定义
- 卡片尺寸：120px ~ 240px 自由调节
- 卡片透明度：50% ~ 100% 自由调节

### 📍 窗口定位
底部 / 顶部 / 左侧 / 右侧 / 居中，多种位置预设。

### 🔒 本地存储
全部数据仅存本地 `JSON + PNG`，不上传云端，隐私无忧。

### 🔗 链接与文件
- 链接直接点击打开
- 文件点击直达所在文件夹

### ✏️ 内容编辑
文本 / 链接 / 代码类型支持直接编辑修改。

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + 1` | 显示/隐藏主窗口 |
| `Ctrl + 2` | 启动截图 |
| `Ctrl + 3` | 快速插入文本 |
| `Esc` | 关闭截图 / 关闭贴图 |

所有快捷键均可在设置页面自定义修改。
